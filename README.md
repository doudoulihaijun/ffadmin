##技术栈：
	vue-cli<br>
	element-ui
	vue
	html+css
	java
	springboot2.0
	等等

##用途
（1）新手可以用来学习前端vue与element的知识；
（2）后台人员可以用来了解前端知识，并为自己的产品搭建框架；
（3）企业可以快速使用框架搭建自己的后台管理系统，用来相应需求；
（4）极简的框架，让前端与后台人员有更大的自由发挥空间；
